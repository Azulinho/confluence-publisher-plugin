The generated content will be inserted between the START and END markers. Both markers must exist in the page. Any existing content between the markers will be replaced.

Example:

  • Start Token = "%%START%%"
  • End Token = "%%END%%"
  • Generated text = "2011-06-19"
  • Existing content:
    Lorem ipsum
%%START%%
previous content
%%END%%
Ipsum lorem
  • New content:
    Lorem ipsum
%%START%%
2011-06-19
%%END%%
Ipsum lorem
